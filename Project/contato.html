Termos e Condições

1. Coleta e Uso de Dados

Ao enviar este formulário, você autoriza a Incentivart a coletar e processar os dados fornecidos para:

  • Responder sua solicitação de contato
  • Desenvolver propostas para seu projeto cultural
  • Enviar informações relevantes sobre nossos serviços
  • Manter histórico de comunicação para melhor atendimento

2. Proteção de Dados

Seus dados pessoais são protegidos conforme a Lei Geral de Proteção de Dados (LGPD). Garantimos:

  • Armazenamento seguro das informações
  • Não compartilhamento com terceiros sem autorização
  • Acesso restrito apenas à equipe autorizada
  • Exclusão dos dados mediante solicitação

3. Comunicação

Ao fornecer seus dados de contato, você autoriza:

  • Contato via e-mail para resposta da sua solicitação
  • Contato via WhatsApp/telefone quando necessário
  • Envio de propostas e orçamentos relacionados ao seu projeto
  • Comunicações sobre oportunidades de editais relevantes

4. Confidencialidade

Todas as informações sobre seu projeto cultural são tratadas com absoluta confidencialidade. Comprometemo-nos a:

  • Não divulgar detalhes do seu projeto para terceiros
  • Manter sigilo sobre informações estratégicas
  • Usar as informações apenas para desenvolvimento da proposta

5. Direitos do Titular

Você possui os seguintes direitos sobre seus dados:

  • Confirmar a existência de tratamento dos dados
  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou incorretos
  • Solicitar exclusão dos dados
  • Revogar o consentimento a qualquer momento

6. Contato

Para exercer seus direitos ou esclarecer dúvidas sobre o tratamento de dados:
